main

voltar

A tag main é onde fica o conteúdo principal da página do
website, é uma tag semantica e so pode ser usada uma por página

section

voltar

Tag semantica que indica a seção dentro do conteúdo
principal no caso o main dentro do main pode ter varias sections

aside

voltar

Assim como section é uma seção dentro do main
a tag aside é uma seção também, mais sendo
mais espefico ela é um assunto axilar do assunto principal.

Exemplo: O main fala de carros o aside vai falar de aluguéis de carros
